# TP0 - Tests Unitaires avec JUnit 5

Ce dépôt contient les solutions des exercices de tests unitaires avec JUnit 5.

## Structure du projet

- `src/main/java` : Contient les classes à tester
- `src/test/java` : Contient les classes de test

## Exercices réalisés

1. Classe Person
2. Classe Stack
3. Classe Fibonacci
4. Classe Prime
5. Classe Factorial
